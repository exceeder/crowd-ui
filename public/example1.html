Registering your corporation with the exchange
To register your corporation, you would need to use request-response pattern over a pub-sub. Following needs to happen:
  1. Generate a unique secret request ID for a one-time response topic, e.g. 12345
  2. Subscribe to "exchange.registry.12345"
  3. Post to "exchange.registry" a JSON of: 
    {slot:'superhero', name:'Some Corp.', reqId:12345}
  4. Receive on "exchange.registry.12345" a message in form 
             {
          slot: 'superhero',
          name: 'Some Corp.',
          balance: 10000,
          units: 0,
          lastSold: 0,
          lastPrice: 0,
          key: 'abc12def'
        }
